- Некоторые бесплатные аналоги сервиса Gamma.app:  
Kampus (Slides Copilot). Сервис повторяет основную идею Gamma — создание презентации «под ключ» по текстовому запросу. Работает на русском языке, оплата возможна картой РФ, без VPN. 

StudyAI. Универсальный сервис, который помимо слайдов генерирует изображения и тексты, обрабатывает фото, создаёт баннеры и обложки. Работает из России без VPN, на русском, с бесплатным стартом. 

DiaClass. Онлайн-редактор презентаций с нейросетью, поддерживает русский и другие языки. Есть большое количество шаблонов оформления, генерация текста и изображений. 

Slider AI. Российский генератор презентаций с помощью нейросетей, помогает создавать текстовые слайды, а также диаграммы и 3D-модели

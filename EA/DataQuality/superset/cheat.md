### cheat sheet
- [Как создать дашборд в Superset: гайд для новичков и полезные лайфхаки](https://habr.com/ru/companies/okko/articles/859140/)Чтобы создать отчет, нужно сделать три простых действия: 
  - Создать датасет с данными.
  - Построить визуализацию на этих данных.
  - Отформатировать дашборд. 
- [Apache Superset. Первый взгляд на BI инструмент](https://habr.com/ru/articles/681228/) Базовый алгоритм работы с Superset можно описать четырьмя шагами.
  - Шаг 1. Настроить коннект к БД (Databases).
  - Шаг 2. Подключить физические таблицы / представления к “витринам” платформы (Datasets).
  - Шаг 3. Сформировать в режиме виртуального конструктора на основе датасетов отдельные графики и диаграммы (Charts).
  - Шаг 4. Создать новую управленческую панель путем простого перетаскивания созданных элементов (Dashboards).  
Работать напрямую с файлами txt, csv, xlsx нельзя. Нужно предварительно загружать информацию в БД и только потом писать SQL запросы. Заливка информации возможна прямо из интерфейса, но нужно разрешить данную операцию в настройках БД. Инструментов для предварительной обработки сырых данных нет. Поэтому быстрая ad hoc аналитика со сбором данных из разрозненных плохо структурированных файлов с помощью Superset крайне затруднена. Так как SQL, по сути, основной язык платформы, то и реализация сложных расчетов на стыке данных из разных датасетов будет также проблематична. Но функциональность языка можно расширить путем использования шаблонов Jinja в запросах.

### excel 
В Apache Superset нельзя напрямую указать ссылку на Excel-файл как на источник данных.   
Для использования Excel-файлов необходимо:  
1) конвертировать файл в формат CSV;
2) загрузить его через меню «Sources» (Источники) -> «Upload CSV». В настройках подключения убедитесь, что включена опция «Enable CSV upload». 

Основные этапы работы:
- Подготовка: Сохраните файл .xlsx как .csv (UTF-8).
- Загрузка: Перейдите в Sources > Upload CSV, укажите имя таблицы, разделитель и загрузите файл.
Параметры: При возникновении ошибок с символами в названиях столбцов, добавьте "paramstyle": "format" в настройках подключения. 

После загрузки, Excel-файл станет доступен как полноценная таблица базы данных, которую можно использовать для создания графиков и дашбордов.

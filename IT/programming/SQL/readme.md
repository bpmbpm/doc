### COUNT_DISTINCT
COUNT(DISTINCT column_name) /особый/ — это SQL-функция, используемая для подсчета количества уникальных (различных) ненулевых значений в заданном столбце.  
В отличие от обычного COUNT, она удаляет повторяющиеся значения перед подсчетом, позволяя узнать число уникальных элементов.

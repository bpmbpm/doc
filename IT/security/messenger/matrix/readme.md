#### keet
(дружелюбнее того же matrix)

Эта система написана на JavaScript с низким уровнем на С. Как и все приложения на JavaScript оно будет требовать очень много ресурсов и медленно работать.
Сервера для Matrix сейчас пишут на Rust (нет сборщика мусора).
Плюс, keet - это очередной стартап, который скорее всего скоро умрет из-за текущего кризиса. Они ничего необыкновенного не делают, так что вряд ли кто-то захочет их купить. 
Но как временное пристанище они могут быть полезны. Если, конечно, данные можно легко экспортировать, и, по возможности, делать bridge с другими мессенджарами.

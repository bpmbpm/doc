## PL_SPARQL
search: Процедурное расширение для SPARQL  
по аналогии с [PL\SQL](https://habr.com/ru/articles/862438/) (Procedural Language/Structured Query Language)   

Примеры функциональных расширений:
- GeoSPARQL: Работа с геопространственными данными.
- RDFRank: Ранжирование сущностей.
- Virtusio: используется процедурный язык PL\SQL - подобного типа для работы с данными
- Extentions to SPARQL

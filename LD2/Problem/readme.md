
### https://en.wikipedia.org/wiki/SHACL
Расширения JavaScript SHACL [ 5 ] (заметка рабочей группы W3C) определяют, как JavaScript можно использовать для выражения ограничений, правил, функций и других возможностей. 
Это охватывает аналогичные аспекты, что и SHACL-SPARQL, но с использованием JavaScript в качестве языка выполнения.

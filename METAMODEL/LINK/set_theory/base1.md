## Что означает нотация L → L² в теории множеств?

> **Кратко:**  
> В теории множеств выражение **L → L²** обозначает множество всех функций, которые каждому элементу множества L сопоставляют упорядоченную пару элементов из L (то есть элемент из L² = L × L). Это стандартная запись для пространства функций между множествами.

---

### Подробное объяснение

#### 1. Что такое L в теории множеств?

- **L** — это обычно обозначение множества. В классической и логической теории множеств L часто означает "конструируемую вселенную" Гёделя, но в общем случае L — просто некоторое множество, например, L = {a, b} или L = ℝ (множество вещественных чисел) .

#### 2. Что такое L²?

- **L²** — это обозначение для декартова квадрата множества L, то есть L × L.
- Формально:  
  L² = L × L = { (x, y) | x ∈ L, y ∈ L }
- Пример:  
  Если L = {1, 2}, то  
  L² = { (1,1), (1,2), (2,1), (2,2) } , , .

#### 3. Что означает стрелка →?

- **→** — это стандартное обозначение для множества всех функций (отображений) из одного множества в другое.
- L → L² — это множество всех функций f, таких что для любого x ∈ L, f(x) ∈ L² , .

---

### Примеры

#### Пример 1: Малое конечное множество

Пусть L = {a, b}.

- Тогда L² = { (a,a), (a,b), (b,a), (b,b) }.
- Функция f ∈ L → L² — это правило, которое каждому элементу из L сопоставляет одну из 4 пар из L².

**Сколько таких функций?**  
Для каждого из 2 элементов L есть 4 варианта выбора пары, итого 4 × 4 = 16 различных функций.

**Пример одной функции:**  
- f(a) = (a, b)
- f(b) = (b, a)

---

#### Пример 2: Вещественные числа

Пусть L = ℝ.

- L² = ℝ × ℝ — множество всех упорядоченных пар вещественных чисел (то есть плоскость).
- Функция f: ℝ → ℝ² может быть, например, f(x) = (x, x²).  
  Тогда f(2) = (2, 4), f(-1) = (-1, 1).

---

#### Пример 3: Диагональное отображение

- f(x) = (x, x) — каждому элементу сопоставляется пара, где оба компонента равны x.

---

### Практические применения

- **Комбинаторика:** Подсчёт числа функций из L в L² — классическая задача.
- **Алгебра:** Описание бинарных операций, когда каждому элементу сопоставляется пара (например, для построения графов или автоматов).
- **Информатика:** Моделирование структур данных, где каждому ключу сопоставляется пара значений.

### Ключевой вывод

> **L → L²** — это множество всех функций, которые каждому элементу множества L сопоставляют упорядоченную пару из L.  
> Это фундаментальное понятие для описания отображений, операций и структур в математике и информатике.
